# ecommerce-return-analysis
# Customer Feedback and Return Analysis in E-Commerce

## Project Overview
This project analyzes customer feedback and product return patterns in an e-commerce dataset to identify factors influencing returns and customer dissatisfaction.

## Tools Used
- Microsoft Excel
- Power BI

## Dataset
The dataset contains order details including:
- Product category
- Sales and profit
- Discounts
- Return status
- Customer segment
- Region

## Key Analysis
The project examines:
- Return patterns across product categories
- Regional return trends
- Relationship between discounts and returns
- Impact of returns on profitability

## Files in this Repository
- E-commerce Dataset (Excel)
- Power BI Dashboard (.pbix)
- Project Report

## Key Insights
- Certain product categories have higher return rates.
- Discounts may increase the probability of product returns.
- Return patterns vary across regions.
